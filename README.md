# Atlassian REST Clients

Collection of strongly-typed REST clients for Atlassian Cloud APIs, automatically generated using [Microsoft Kiota](https://github.com/microsoft/kiota) from official OpenAPI specifications.

## 📦 Packages

### 🔷 Jira REST Client v2

Complete REST client for **Atlassian Jira Cloud API v2**. Includes support for projects, issues, workflows, and more.

## 🚀 Quick Start

Install your desired package:

```bash
# Jira
dotnet add package Kubis1982.Atlassian.Jira.RestClient.v3

# Bitbucket
dotnet add package Kubis1982.Atlassian.Bitbucket.RestClient.v2

# Confluence
dotnet add package Kubis1982.Atlassian.Confluense.RestClient.v2
```

## ✨ Features

- ✅ Fully typed C# clients
- ✅ Generated from official OpenAPI specifications
- ✅ Built with Microsoft Kiota code generation
- ✅ Support for .NET 10.0+
- ✅ Comprehensive API coverage

## 📋 OpenAPI Specifications

Each package is generated from official Atlassian OpenAPI specifications:

- **Jira v3:** `https://dac-static.atlassian.com/cloud/jira/platform/swagger-v3.v3.json?_v=1.8453.0`
- **Bitbucket v2:** `https://dac-static.atlassian.com/cloud/bitbucket/swagger.v3.json?_v=2.300.163`
- **Confluence v2:** `https://dac-static.atlassian.com/cloud/confluence/openapi-v2.v3.json?_v=1.8453.0`
